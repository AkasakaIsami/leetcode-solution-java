## My Solution

​	我的解法，暴力嵌套循环（时间复杂度O(n<sup>2</sup>)），喜闻乐见的超时。看了下超时的测试用例，官方真是丧心病狂。看答案有人说在循环前加一点条件判断就可以不超时，学习了一下。主要是用条件判断规避了一些不必要的二层循环，就是如果以当前的高度height[i]，加水到最宽都没有现有的最大面积大，就不用算这个循环了。
