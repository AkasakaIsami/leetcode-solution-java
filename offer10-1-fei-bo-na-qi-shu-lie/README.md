# Offer10- I. 斐波那契数列
- 网址：https://leetcode-cn.com/problems/fei-bo-na-qi-shu-lie-lcof/

- 日期：2022.03.26

  

## Description

写一个函数，输入 `n` ，求斐波那契（Fibonacci）数列的第 `n` 项（即 `F(N)`）。




## What's new
- Fibonacci：斐波那契数列，每一项是前两项之和
- 动态规划：DP



## My Solution

第一版是简单递归，但发现会超出限制。第二版是记忆化递归，用数组存储了已经计算过的项。

最优解是DP
